# 2-player-game-of-war
An exercise in java with playing cards


To run, first make sure you have java installed, then, in cmd, open project repository:

      cd #project_repo_path
      
then, compile files:

      javac card.java
      javac deck.java
      javac hand.java
then run Hand.java:

      java hand.java

output should look like this:


      Hand 1:

      the 8 of Hearts
      the 7 of Spades

      Hand 2:

      the King of Hearts
      the 9 of Clubs


      hand 2 wins
